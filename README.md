
# Harduf Global Investments Website

This is the official landing page for **Harduf Global**, a company with over 15 years of experience in international investment, real estate, and strategic financial ventures.

## Features
- Elegant, modern design
- Hebrew content and right-to-left layout
- Background and logo integration
- Accessibility tools for enhanced user experience
- Mobile-friendly responsive layout

## Deployment
This site is ready to be deployed using GitHub Pages.  
Make sure the `CNAME` file points to your custom domain: `harduf-global.com`.
